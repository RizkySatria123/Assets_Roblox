# Assets_Roblox

Repository ini berisi koleksi aset 3D dan script untuk keperluan pengembangan game (khususnya Roblox).

## Daftar Aset


## Cara Penggunaan Script

1. Buka software **Blender**.
2. Masuk ke workspace **Scripting**.
3. Buka file `Asset Rizky/Gapura.Pyt` atau copy-paste kodenya ke text editor Blender.
4. Tekan tombol **Run Script**.
5. Model Gapura akan digenerate secara otomatis di titik pusat (0,0,0).

## Struktur Folder

- `Asset Rizky/`: Folder berisi script Python untuk pembuatan aset prosedural.
